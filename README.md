### How to use it?

First install npm packges
```shell
npm install
#or
yarn
```

After this, you can run localy or build an exportable version
```
# run localy
npm start

# build exportable version
# for linux
npm run package-linux

# Mac
npm run package-mac

# Windows
npm run package-win
```
